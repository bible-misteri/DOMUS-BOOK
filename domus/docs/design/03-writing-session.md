# 03 — Writing Session

Status: Draft v1.0

---

# Tujuan

Writing Session adalah ruang kerja utama seorang penulis.

Di sinilah sebuah buku benar-benar ditulis.

Seluruh desain harus membantu penulis tetap fokus tanpa gangguan.

---

# Filosofi

DOMUS tidak membuka Editor.

DOMUS membuka sebuah sesi menulis.

Setiap sesi memiliki satu tujuan yang jelas.

Penulis hanya perlu memikirkan satu hal:

"Tulis."

---

# Layout

+------------------------------------------------------------+

DOMUS ISAACI

Bab 1

Mengapa Ishak Dilupakan?

--------------------------------------------------------------

Estimasi

45 menit

--------------------------------------------------------------

[ MULAI MENULIS ]

+------------------------------------------------------------+

---

# Tombol

## MULAI MENULIS

Membuka Editor.

DOMUS langsung membuka bab terakhir yang sedang dikerjakan.

Cursor berada tepat di posisi terakhir sebelum aplikasi ditutup.

Tidak ada dialog tambahan.

Tidak ada pemilihan file.

---

# Informasi

Halaman hanya menampilkan:

- Judul Buku
- Judul Bab
- Target hari ini
- Estimasi waktu

Tidak menampilkan:

- Folder
- Explorer
- Markdown
- Metadata
- Struktur proyek

---

# Aturan UI

- Maksimal satu tombol utama.
- Fokus visual pada tombol "Mulai Menulis".
- Tidak ada panel samping.
- Tidak ada AI yang muncul otomatis.

---

# Target UX

Pengguna membuka DOMUS.

↓

Klik "Tulis Buku"

↓

Klik "DOMUS ISAACI"

↓

Klik "Mulai Menulis"

↓

Editor terbuka.

Seluruh proses harus selesai kurang dari 20 detik.
