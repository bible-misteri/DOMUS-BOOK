# 02 — My Books

Status: Draft v1.0

---

# Tujuan

Halaman ini menampilkan seluruh buku milik pengguna.

Pengguna harus dapat:

- melanjutkan buku yang sedang ditulis;
- membuat buku baru;
- menghapus buku yang tidak diperlukan.

Halaman ini tidak menampilkan pengaturan teknis proyek.

Fokusnya adalah buku.

---

# Filosofi

Seorang penulis berpikir dalam "buku", bukan "folder".

Karena itu DOMUS tidak menampilkan struktur file.

DOMUS hanya menampilkan karya yang sedang dikerjakan.

---

# Layout

+------------------------------------------------------+

BUKU SAYA

--------------------------------------------------------

📖 DOMUS ISAACI

Progress : 48%

Terakhir dibuka : Hari ini

[ LANJUT MENULIS ]

--------------------------------------------------------

➕ BUAT BUKU BARU

+------------------------------------------------------+

---

# Tombol

## 📖 DOMUS ISAACI

Membuka halaman Project.

File berikutnya:

03-writing-session.md

---

## ➕ BUAT BUKU BARU

Menjalankan Wizard pembuatan buku.

Wizard terdiri dari:

1. Judul Buku
2. Penulis
3. Jenis Buku
4. Selesai

Setelah selesai, DOMUS langsung membuka Writing Session.

---

# Informasi Buku

Setiap kartu buku menampilkan:

- Judul
- Progress
- Terakhir dibuka
- Tombol "Lanjut Menulis"

Tidak menampilkan:

- Folder
- Metadata YAML
- Markdown
- Nama file

---

# Aturan UI

- Satu kartu untuk satu buku.
- Maksimal satu tombol utama pada setiap kartu.
- Progress menggunakan progress bar sederhana.
- Layout harus tetap bersih meskipun jumlah buku banyak.

---

# Target UX

Pengguna baru harus dapat:

1. Membuka DOMUS.
2. Memilih buku.
3. Melanjutkan menulis.

Kurang dari 15 detik.

Jika lebih lama, desain harus disederhanakan.
